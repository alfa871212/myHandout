\chapter{波}
\section{波的性質}
\subsection{定義}
\begin{enumerate}
    \item 波：物質受力後產生上下起伏或疏密相間的現象
    \item 波動：物體受外力擾動，引發鄰近物質被擾動，稱為波動；一開始被擾動的地方稱為波源
\end{enumerate}
\subsection{特性}


\begin{enumerate}
    \item 物質振動時，具有能量，故波是一種能量的展現
    \item 波只傳遞能量，不傳遞介質
    \begin{note}{結論？}
    \end{note}
\end{enumerate}
